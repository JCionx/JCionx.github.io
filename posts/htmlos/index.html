htmlOS

May 19, 2023

Hello there!

I am currently developing htmlOS!

htmlOS is supposedly an operating system made entirely with HTML, CSS and JavaScript.

I made a simple function that allows windows to be spawned in the HTML’s body. Then I decided to make a taskbar, a menu, and some apps. Now there are even notifications, APIs, and an App Store.

The usage of the API is simple. Open the settings app inside htmlOS, and use the copy code button. In your htmlOS app, insert this:

// Example notification API usage
parent.shownotification('Title, 'Description', '3000')

// You need to include "parent" before any API call, because your htmlOS app will run inside an iframe.

If you want to try htmlOS for yourself, do so by clicking on this link.

That’s it!
